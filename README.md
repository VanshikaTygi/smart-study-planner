# Smart Study Planner (Rule-Based)

## 📌 Overview
Smart Study Planner is a Python-based, rule-driven productivity system designed to help students track study sessions, analyze subject-wise effort, and receive intelligent, explainable feedback to improve consistency and balance.

The project focuses on **rule-based intelligence** rather than black-box machine learning, making the logic transparent, interpretable, and easy to extend.

---

## ⚙️ Features
- Study and revision session tracking
- Subject-wise time aggregation
- Smart rule-based insights
- Productivity score (0–100)
- Consistency and balance feedback
- Menu-driven CLI interface

---

## 🧠 Intelligence Used
- Rule-based decision system
- Threshold-based recommendations
- Behavioral analysis using aggregated study data
- Explainable logic (no black-box ML)
